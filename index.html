Mongo DB

MongoDB is a popular open-source NoSQL database designed for scalability, flexibility, and high performance. Unlike traditional relational databases, MongoDB uses a document-oriented model, storing data in JSON-like format called BSON (Binary JSON).

node.js

Node.js is a powerful, open-source, server-side runtime environment built on Chrome's V8 JavaScript engine. It allows developers to execute JavaScript code outside the browser, making it ideal for building scalable and high-performance server-side applications.

react

React is a popular, open-source JavaScript library developed by Facebook for building user interfaces, particularly single-page applications. It enables developers to create reusable UI components and efficiently manage the state of web applications.
